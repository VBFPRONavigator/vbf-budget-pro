Vigência: 25 de janeiro de 2026
Última atualização: 25 de janeiro de 2026

Visão Geral

O BudgetPro Navigator está empenhado em proteger a sua privacidade. Esta política explica as nossas práticas de dados para a aplicação de desktop BudgetPro Navigator ("a App").

Armazenamento de Dados

Todos os dados são armazenados localmente no seu dispositivo. O BudgetPro Navigator NÃO faz o seguinte:

  • Enviar dados para servidores externos
  • Fazer upload de informações para a nuvem
  • Transmitir os seus dados financeiros pela Internet
  • Partilhar as suas informações com terceiros

Segurança

  • Apenas Local: Os dados permanecem encriptados no seu hardware.
  • Sem Dependência da Internet: A aplicação funciona 100% offline.
  • Em Conformidade: Cumprimos com o RGPD, CCPA e COPPA.

Resumo: Os seus dados financeiros são apenas seus. Não os vemos, não os armazenamos e não os partilhamos.
