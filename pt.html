Política de Privacidade

Data de vigência: 25 de janeiro de 2026
Última atualização: 25 de janeiro de 2026

1. Visão Geral

O BudgetPro Navigator está empenhado em proteger a sua privacidade. Esta política de privacidade explica as nossas práticas de dados para a aplicação de desktop BudgetPro Navigator ("a App").

2. Armazenamento de Dados

Todos os dados são armazenados localmente no seu dispositivo. O BudgetPro Navigator não realiza nenhuma das seguintes ações:

  • Envio de dados para servidores externos
  • Upload de informações para a nuvem
  • Transmissão dos seus dados financeiros através da Internet
  • Partilha das suas informações com terceiros

3. Recolha de Dados

Não recolhemos NENHUM dado. A App funciona inteiramente offline e não recolhe, transmite nem armazena qualquer informação do utilizador fora do seu dispositivo local.

O que não recolhemos:
  • Informações pessoais (nome, e-mail, número de telefone)
  • Credenciais de contas financeiras
  • Dados de transações
  • Dados de localização
  • Análises de utilização
  • Relatórios de erro (crash reports)
  • Dados de telemetria

4. Segurança dos Dados

Os seus dados financeiros são protegidos por:

  • Armazenamento apenas local: Todos os dados permanecem no seu dispositivo de forma encriptada.
  • Sem dependência da Internet: A App funciona completamente offline.
  • Sem acesso de terceiros: Nenhum serviço externo ou API tem acesso aos seus dados.
  • Controlo do utilizador: Você tem o controlo total sobre os seus dados e pode eliminá-los a qualquer momento.

5. Direitos do Utilizador

  • Acesso: Consulte todos os seus dados dentro da App a qualquer momento.
  • Exportação: Crie cópias de segurança utilizando a função integrada.
  • Eliminação: Elimine todos os dados desinstalando a App ou através da função "Limpar todos os dados".
  • Controlo: Gira todos os aspetos da sua informação financeira sem supervisão externa.

6. Serviços de Terceiros

Não utilizamos NENHUM serviço de terceiros. A App não inclui ferramentas de análise, redes publicitárias, nem se liga a redes sociais ou serviços de armazenamento na nuvem.

7. Privacidade das Crianças

A App foi concebida para adultos. Não recolhemos intencionalmente informações de crianças menores de 13 anos.

8. Notificação de Violação de Dados

Uma vez que nenhum dado é transmitido ou armazenado fora do seu dispositivo, não existe risco de uma violação de dados a partir dos nossos servidores.
